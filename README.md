# Interior Design Studio Website

A modern, minimalist static website for an interior design business focused on residential clients.

## Features

- **Homepage**: Hero section with call-to-action buttons
- **Portfolio**: Gallery showcasing completed projects
- **About**: Designer background and philosophy
- **Services**: Detailed service offerings and process
- **Blog**: Design tips and inspiration articles
- **Contact**: Simple contact page with email

## Technology

- HTML5
- CSS3 (vanilla, no frameworks)
- Fully responsive design
- Mobile-first approach

## Setup

1. Clone this repository
2. Add your images to the appropriate folders in `/images`
3. Update content in HTML files with your information
4. Customize colors and styling in `css/styles.css`

## Image Requirements

Place images in the following directories:
- `/images/hero/` - Hero background image
- `/images/portfolio/` - Project images (project-1.jpg through project-6.jpg)
- `/images/about/` - Designer photo (designer.jpg)
- `/images/blog/` - Blog post images

Recommended image sizes:
- Hero: 1920x1080px
- Portfolio: 800x600px
- Blog: 800x500px

## GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select branch (usually `main`) and root folder
4. Save and wait for deployment
5. Your site will be available at `https://yourusername.github.io/repository-name`

## Customization

### Update Contact Information
Edit `contact.html` and update the email address and service area information.

### Change Colors
Modify the CSS variables and color values in `css/styles.css` to match your brand.

### Add More Blog Posts
1. Create a new HTML file in the `/blog` directory
2. Use existing blog posts as templates
3. Add a new blog card to `blog.html`
