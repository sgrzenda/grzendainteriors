# Image Directory

Add your images to the appropriate subdirectories:

## Required Images

### Hero
- `hero/hero-bg.jpg` - Main hero background (1920x1080px recommended)

### Portfolio
- `portfolio/project-1.jpg` through `portfolio/project-6.jpg` (800x600px recommended)

### About
- `about/designer.jpg` - Professional photo of the designer (800x800px recommended)

### Blog
- `blog/minimalist-living.jpg` - Featured image for minimalist living post
- `blog/color-psychology.jpg` - Featured image for color psychology post
- `blog/small-spaces.jpg` - Featured image for small spaces post

## Image Guidelines

- Use high-quality images
- Optimize for web (keep file sizes under 200KB when possible)
- Use JPG format for photos
- Maintain consistent aspect ratios within each category
- Ensure images are properly licensed for use

## Placeholder Images

Until you have your own images, you can use placeholder services like:
- https://unsplash.com (free high-quality photos)
- https://placeholder.com (simple placeholders)
